# ASPNETCore7-JWT-BearerEvents-Swagger-Extensions-HttpFiles_ContagemAcessos
Exemplo de API REST para contagem de acessos criada com o .NET 7 + ASP.NET Core, empregando extensões definidas em uma Class Library para utilização de JWT (JSON Web Tokens) + JwtBearerEvents e de configurações para que o Swagger suporte tokens. Inclui arquivos .http para testes a partir do VS Code e do Visual Studio 2022.
